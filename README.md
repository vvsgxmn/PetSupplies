## 项目介绍

宠物用品租赁系统是一个基于 Spring Boot + Vue 前后端分离的 Web 应用，面向宠物主人提供用品共享租赁服务。用户可浏览各驿站的宠物用品并在线下单租赁，管理员可在后台管理驿站、物品、分类、库存、清洁流程和订单。

## 功能模块

- **用户端首页**：展示热门物品、驿站分布，支持快捷浏览
- **物品浏览与详情**：按分类/驿站筛选物品，查看物品详情及租赁信息
- **在线租赁下单**：选择物品、租赁天数，自动计算租金和押金
- **我的订单**：查看订单状态、租赁进度
- **个人中心**：修改昵称、头像、密码
- **管理后台 - 数据看板**：统计物品总数、订单量、租赁趋势图、分类分布图
- **管理后台 - 驿站管理**：驿站增删改查
- **管理后台 - 物品管理**：物品上下架、编辑、库存操作
- **管理后台 - 分类管理**：物品分类维护
- **管理后台 - 库存管理**：库存变动日志、入库出库
- **管理后台 - 清洁管理**：清洁记录跟踪与审核
- **管理后台 - 订单管理**：全部订单查看与详情

## 技术架构

- **后端**：Spring Boot 2.6.13 + MyBatis-Plus 3.5.3.1 + JWT 认证
- **前端**：Vue 3.5 + Element Plus 2.4 + Vite 8 + ECharts 5.4 + Pinia
- **数据库**：MySQL 8.0
